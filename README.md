# Swastik OS

**Swastik OS** is a personal portfolio website reimagined as a fully functional **Mac OS 9 (Platinum)** desktop environment. Built with modern web technologies, it offers a nostalgic yet interactive way to explore my skills, projects, and professional background.

## üåü Features

-   **Authentic Mac OS 9 UI**: Meticulously recreated Platinum design language, including window borders, typography, and icons.
-   **Window Manager**: Fully interactive windows that can be dragged, resized, minimized, and maximized.
-   **Desktop Environment**:
    -   **Desktop Icons**: Draggable icons for navigation.
    -   **Dock**: An interactive dock with magnification effects for quick app access.
    -   **Top Bar**: Functional menu bar with clock and system menus.
-   **Applications**:
    -   **About Me**: A profile overview.
    -   **Resume**: Integrated PDF viewer to preview and download my resume.
    -   **Projects**: Fetches and displays my top repositories directly from the GitHub API.
    -   **Skills**: Visual breakdown of my technical expertise.
    -   **Contact**: Easy ways to get in touch.
-   **Boot Sequence**: A retro boot-up animation to set the mood.

## üöÄ Run Locally

1.  **Clone the repository**
    ```bash
    git clone https://github.com/SwastikShetty06/swastik-os.git
    cd swastik-os
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open in your browser**
    Visit `http://localhost:5173` to experience Swastik OS.
